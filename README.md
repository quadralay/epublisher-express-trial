# ePublisher Express Trial

Documentation and sample project for the WebWorks ePublisher Express trial experience.

## Overview

This repository contains:

- **Trial Guide** - Quick-start guide for new Express trial users
- **Sample Project** - Markdown++ source documents demonstrating ePublisher capabilities

## Project Structure

```
epublisher-express-trial/
├── latest/                          # Current versions
│   ├── local-express-trial-project/ # Sample ePublisher project
│   └── online-trial-guide/          # Online quick-start guide
├── legacy/                          # Archived previous versions
└── docs/                            # Planning and prompt history
```

## Markdown++

Source documents are authored in [Markdown++](https://static.webworks.com/docs/epublisher/latest/help/Authoring%20Source%20Documents/_markdown.1.01.html), a backward-compatible extension of Markdown with additional features for professional documentation:

- Content islands (callouts, notes, warnings)
- Multiline tables
- Conditional content
- Include directives
- Markers for metadata (Keywords, Description)
- Anchors for cross-references
